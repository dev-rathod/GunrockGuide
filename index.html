01 — About the Project

What is
GunrockGuide?

GunrockGuide is an embedded systems navigation tool built to solve a real problem every UC Davis student faces — getting lost inside Kemper Hall's sprawling corridors.

Using a GPS module, IR receiver, and OLED display connected to a CC3200 microcontroller, you type in your destination room number using a remote. The device sends your GPS coordinates to AWS Lambda, which computes the shortest path and emails you a custom map image showing exactly how to get there.

Built as the EEC 172 final project, every component — from hardware design to cloud integration — was engineered from scratch by Amal and Dev.

📍

GPS-Based Positioning

A GPS module captures your exact location and sends coordinates to AWS so the system knows where you are inside Kemper Hall.

🔢

IR Remote Input

Type your destination room number on the OLED screen using an IR receiver and remote — no phone or touchscreen needed.

✉️

Email Path Delivery

After submitting your request you receive an email with a custom S3-hosted map image. Green dot = you, red dot = destination, with the route drawn between them.

☁️

Cloud Backend

AWS Lambda handles all routing logic and generates the path image on demand, served instantly via Amazon S3.

03 — Development Story

Our
Journey

Phase 1

Problem Discovery

We both experienced the frustration of being late to class because Kemper Hall is impossible to navigate. We decided to solve it with embedded systems.

Phase 2

Hardware Design

Selected the CC3200 microcontroller for its built-in Wi-Fi capabilities. Integrated a GPS module for live positioning, an IR receiver for room-number input, and an OLED display for the user interface.

Phase 3

Software & Routing

Built the AWS Lambda backend to receive GPS coordinates, run pathfinding logic through Kemper Hall's floor plan, generate a route map image in S3, and trigger an email delivery to the user — all within seconds.

Phase 4

Integration & Testing

Brought hardware and software together. Ran live tests inside Kemper Hall — typed in room 1155 using the IR remote, sent the GPS coordinates, and received the correct path image by email in seconds.

Phase 5

Final Demo

Polished the user interface, documented the system, and presented GunrockGuide as our EEC 172 final project. Future work: expand to more buildings and add voice output.

04 — Cloud Infrastructure

AWS
Integration

GunrockGuide's cloud backend is built on Amazon Web Services, enabling real-time path computation and email delivery triggered directly from the embedded device.

λ

AWS Lambda

Serverless functions receive GPS coordinates and the destination room number from the CC3200. Lambda runs pathfinding logic through Kemper Hall's floor plan and generates the route map image stored in S3.

Serverless Functions
📡

IR Receiver & GPS Module

The IR receiver lets you type a destination room number on the OLED display using a remote. The GPS module simultaneously captures your exact coordinates and transmits them to AWS — no phone required.

Real-Time Sensors
✉️

Email Path Delivery

Once Lambda computes the route, it sends you an email containing a custom S3 link. Opening it shows the Kemper Hall floor map with a green dot for your current location, a red dot for your destination, and the path drawn between them.

Path Notification
🪣

Amazon S3

Generated path images are stored in S3 and served via a unique URL included in the delivery email. Floor plan assets for the web interface are also hosted here with low-latency access from anywhere on campus.
